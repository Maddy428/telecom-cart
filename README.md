# Telecom Cart API

A thin Experience API that powers a telecom cart on top of a non-persistent Salesforce cart context.  
Implemented in Node.js and TypeScript using Express with in-memory storage.


## Features

- Create a cart
- Get cart details
- Add item to cart
- Remove item from cart
- Cart expiry implemented (1 hour)
- No database (in-memory store)
- Unit and integration tests included


## Requirements

- Node.js 20+
- npm


## Install Dependency
    npm install

##Run 
    npm run start
